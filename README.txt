── הגדרה ראשונה (פעם אחת בלבד) ──────────────

1. פתח Terminal בתיקייה זו והרץ:
      npm install
      npx playwright install chromium

2. הפעל כניסה ראשונה:
      npm run login
   (דפדפן ייפתח — התחבר ל-mypips, אז לחץ Enter)

3. הגדר משימות אוטומטיות:
   לחץ ימני על setup-tasks.ps1 → "Run with PowerShell"


── שימוש יומי ────────────────────────────────

• פתח את השרת: לחץ פעמיים על start-server.bat
• פורטל: http://localhost:3000
• הנתונים מתעדכנים אוטומטית ב-08:00, 13:00, 19:00

── רענון ידני ────────────────────────────────

   npm run refresh


── קבצים ────────────────────────────────────

  server.js        — שרת Express (פורטל + API)
  refresh.js       — רענון נתונים ממypips
  first-login.js   — כניסה ראשונה לmypips
  portal/          — ממשק הפורטל
  data/            — נתונים שמורים + session
